# 🎓 School Management System

![PHP](https://img.shields.io/badge/PHP-8.2-777BB4?style=for-the-badge&logo=php&logoColor=white) ![Laravel](https://img.shields.io/badge/Laravel-10-FF2D20?style=for-the-badge&logo=laravel&logoColor=white) ![MySQL](https://img.shields.io/badge/MySQL-005C84?style=for-the-badge&logo=mysql&logoColor=white)

> A comprehensive ERP solution designed to streamline administrative tasks, manage academic records, and bridge the gap between students, teachers, and admins.

---

## 🚀 Key Features

| Feature | Description |
| :--- | :--- |
| **🔐 RBAC Security** | Multi-level authentication for Admins, Teachers, and Students. |
| **📚 Academic Management** | Dynamic handling of classes, subjects, and complex timetables. |
| **📊 Smart Grading** | Automated calculation of GPAs and report card generation. |
| **✅ Attendance** | Real-time tracking system for daily attendance. |

---

## 🛠️ Technical Architecture

This project is built using a robust monolithic architecture focusing on data integrity and security.

- **Backend Framework:** Laravel (PHP)
- **Frontend:** Blade Templates with Bootstrap
- **Database:** MySQL (Relational Schema)
- **API Security:** Laravel Sanctum
